# MetalGraph Releases

This repository hosts releases and update feeds for [MetalGraph](https://metal.graphics/app).

## Download

Download the latest version from the [Releases](https://github.com/victorBaro/metalgraph-releases/releases) page.

## About MetalGraph

MetalGraph is a visual shader editor for macOS that lets you create Metal fragment shaders using a node-based interface. It's designed as a companion tool for the [metal.graphics](https://metal.graphics) course.
