## About MetalGraph

MetalGraph is a visual shader editor for macOS and iPad that lets you create Metal fragment shaders using a node-based interface. It's designed as a companion tool for the [metal.graphics](https://metal.graphics) course.

### Features

- **Node-based shader editor** - Connect nodes to build complex shaders visually
- **Real-time preview** - See your shader update as you edit
- **30+ built-in nodes** - Math, color, input, geometry, and more
- **Custom nodes** - Create reusable shader components
- **For loops** - Iterate with composite loop nodes
- **MSL code export** - Copy generated Metal Shading Language code
- **Cross-platform** - Works on macOS and iPad
